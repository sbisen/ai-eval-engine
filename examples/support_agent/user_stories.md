# User Stories — Acme Support Agent

- As a customer, I want to know where my order is so I can plan for delivery.
- As a customer, I want to start a return so I can send back an item that didn't fit.
- As a customer, I want a refund for a damaged item.
- As a customer, I want to update the email on my account.
- As a customer, I want to ask whether a product is in stock before ordering.
